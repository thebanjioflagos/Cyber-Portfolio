# 🛡️ OLABANJI OKUNOLA | Cyber-Portfolio v2.0
> **High-Performance Strategic Assets for Senior Security Engineering & Fullstack Development.**

This is a production-grade, security-hardened portfolio engineered with **Next.js 15**, **Tailwind CSS v4**, and **Framer Motion**. It serves as a unified intelligence terminal showcasing specialized expertise in SIEM/SOC operations, PenTesting, and Secure Product Development.

## 🏗️ Technical Stack
- **Framework**: Next.js 15 (App Router / React 19)
- **Styling**: Tailwind CSS v4 + OKLCH Color Geometry
- **Animations**: Framer Motion (Orchestrated Layout Transitions)
- **Data Layer**: Zero-Database Markdown Architecture (Secure-by-Design)
- **Hardening**: OPSEC-compliant link relays, VDP integration, and JSON-LD AIEO optimization.

## 🚀 Key Modules
### 1. Recruiter Mode™
A specialized, 8-role CV selector engine optimized for high-conversion technical hiring. Supports specialized paths for Security Engineers, PenTesters, and CTOs.

### 2. Live Threat Intelligence Dashboard
A reactive network simulation engine demonstrating real-time detection engineering, log normalization (Sysmon), and mitigation logic.

### 3. Strategic Content Hub (Blog)
A Markdown-powered intelligence feed covering **Threat Intel**, **Defensive Ops**, **Offensive Strategy**, and **DevSecOps**.

## 🔒 Security & OPSEC
- **Vulnerability Disclosure Policy**: Live at `/security-policy`.
- **OPSEC Pivot**: All PII (WhatsApp/Contact) is masked via secure handshakes to prevent automated OSINT harvesting.
- **Audit Ready**: Scanned for hardcoded secrets and credentials.
